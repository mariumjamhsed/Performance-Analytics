# Performance-Analytics
IPL Performance Analytics. Analyzed 10+ seasons of IPL cricket data using pandas, matplotlib, and seaborn.
**Key Findings:**
- Team X won Y finals
- Average score increased Z% over seasons
- Toss impact: teams winning toss have X% match win rate
- Top venue for high scores: [venue name]


## Dataset

The datasets used in this project are **not my own**. They were obtained from publicly available sources and are widely used for learning, data analysis, and machine learning practice.
